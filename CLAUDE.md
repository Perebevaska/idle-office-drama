# CLAUDE.md — Idle Office Drama

Рабочий файл для Claude Code. Полная концепция (идея, ниша, монетизация,
дизайн механик) — в [`claude-concept.md`](./claude-concept.md). Здесь —
актуальное состояние кода и что осталось по плану.

---

## Что это

Idle-игра про офис: ты CEO, сотрудники живут сами, ты заходишь читать «что
натворили» и принимаешь решения. Живой нарратив внутри idle-механики.
Сначала — Telegram Mini App, потом — нативный Android.

---

## Стек

```
React 19 + TypeScript + Vite 8 + Tailwind v4
Zustand (состояние) + localStorage (сохранение)
```

**Принцип:** `src/core/` — платформо-независимое ядро (без React/Telegram).
Это позволит перенести логику в React Native / Android без переписывания.

---

## Структура

```
src/
  core/                 ← ядро (чистый TS, без UI)
    types.ts            ← все типы (Employee, EventTemplate, GameState...)
    employees.ts        ← модель, найм/увольнение, статы
    events.ts           ← движок: подбор по условиям, исходы, цепочки
    engine.ts           ← тик мира, офлайн-просчёт, экономика
    templates/events.json ← шаблоны событий (сейчас 10)
  store/gameStore.ts    ← Zustand: тики, найм/увольнение, сейв/валидация
  ui/components/        ← StatBar, EmployeeCard, EventCard, HireSection
  App.tsx               ← экран: лента, команда, найм, история
```

## Запуск

```bash
npm install
npm run dev          # http://localhost:5173 (на VPS: --host уже в vite.config.ts)
npm run build        # tsc -b && vite build
npx tsc --noEmit     # только типы
```

VPS: dev открыт наружу на `http://91.197.0.63:5173/` (ufw 5173 открыт,
HMR-host прописан в `vite.config.ts`). Заглушить: `pkill -f vite`.

**Репозиторий:** https://github.com/Perebevaska/idle-office-drama (push по HTTPS-токену).

## Стиль

Иронично-корпоративный, как офисные мемы. Русские имена (Петя, Маша...),
переключаемые на нейтральные для интернационала. Без grimdark — абсурдно и
смешно. События узнаваемые («это прям мой офис»).

## Разделение ролей

- **Claude Code:** весь TS/React код, движок, шаблоны, UI, интеграции, деплой-конфиги.
- **Владелец:** бот в BotFather, аккаунт Vercel, финальный дизайн, коммьюнити.
